# Contact List - React App

A simple and interactive Contact List application built with React. Users can add new contacts with name and email, and remove existing ones. Perfect for learning React state management with `useState` and dynamic list rendering.

## ✨ Features

- Add new contacts (Name + Email)
- Remove contacts individually
- Real-time list updates
- Clean and responsive UI
- Form validation (prevents empty entries)

## 🛠 Technologies Used

- **React.js** (Functional Components + Hooks)
- `useState` for state management
- JavaScript (ES6+)
- CSS (via App.css)
